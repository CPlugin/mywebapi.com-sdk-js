Manager (live) call. Wraps `UsersGroupOp(GroupCommandInfo, ICollection)`. The body specifies the command and its parameter (NewGroup for SetGroup, Leverage for Leverage; both ignored for Delete/Enable/ Disable) plus the list of target logins. The wrapper auto-fills the internal `Len` field from the logins array — clients do not set it. Wine x64 safe: the wrapper uses `AllocArraySafe` on the int login array (single contiguous pack, no UnpackObject loop) and `AllocSafe` on the GroupCommandInfo struct. Requires Manager or Administrator access rights on the manager account; the wrapper enforces this server-side. Idempotency-Key strongly recommended — bulk Delete / SetGroup operations are destructive on customer-visible state. ## Parameters ### tradePlatform `string` ### mT4UsersGroupOp?
