POST mutator. Pass the full `MT4UserCreate` DTO. Set `Login = 0` to let MT4 server assign the next free id, or request a specific id by setting `Login > 0` (the server rejects collisions with an MT4 error envelope). The wrapper accepts the account with empty password bytes; clients MUST follow up with `POST UserPasswordSet/{login}` before the account is usable. Idempotency-Key strongly recommended — a retried create without it can land twice when the original response was lost on the wire, burning a second login id from the broker's sequence. ## Parameters ### tradePlatform `string` ### mT4UserCreate?
