Pump-cached call. Cancels a pending trade transaction that the MT4 server is still holding in the rollback buffer (e.g. an instant-execution requote that has not yet been confirmed). Has no effect once the transaction has been committed; returns the wrapper's `ResultCode` as part of the envelope on failure.
