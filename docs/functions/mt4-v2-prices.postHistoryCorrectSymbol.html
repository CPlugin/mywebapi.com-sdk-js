Manager (live) call. The MT4 server walks the symbol's full bar history for every period and fixes inconsistencies (gaps, broken OHLC relationships, mismatched aggregates). Returns the count of bars that the server corrected — zero is a valid result (history was already consistent). Requires Administrator rights on the manager account.
