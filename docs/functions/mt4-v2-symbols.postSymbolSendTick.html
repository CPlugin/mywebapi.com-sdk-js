Manager-live POST. The wrapper requires the manager account to hold the `Market Watch` permission; without it the server typically drops the connection rather than returning an error. The API-side `ResourceAccessAuthorize` on this endpoint guards against API callers without the appropriate platform-level permission, but broker-side mis-configuration of the underlying manager rights remains the consumer's responsibility. `bid` and `ask` are absolute prices, not deltas. Pass the symbol's last-known bid/ask if you only need to refresh the timestamp; pass adjusted prices to actually move the quote. ## Parameters ### tradePlatform `string` ### params?
