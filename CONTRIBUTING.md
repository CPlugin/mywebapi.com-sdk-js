# Contributing

Thank you for your interest in contributing to `@mywebapi.com/sdk`! This SDK provides a TypeScript client for the SaaS WebAPI v2 (MT4/MT5 trading platform management).

## How to contribute

1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/my-feature`).
3. Commit your changes (`git commit -m 'Add my feature'`).
4. Push to the branch (`git push origin feature/my-feature`).
5. Open a Pull Request.

## Development

See the [Development](#development) section in README.md for local setup instructions.

## Code style

- TypeScript strict mode is enabled.
- Follow the existing code conventions in the project.
- Run `bun run typecheck` before committing.
